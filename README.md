# Kotlin Practice 

This repository contains my Kotlin learning journey, starting from the fundamentals and progressing through hands-on console applications. Each project helped me strengthen my understanding of Kotlin syntax, object-oriented programming, collections, and problem-solving.

---

##  Topics Covered

- Kotlin Basics
- Variables and Data Types
- Conditional Statements
- Loops
- Functions
- Null Safety
- Object-Oriented Programming
- Data Classes
- Collections
- Exception Handling
